# Credit-scoring-Model
Machine Learning project for predicting creditworthiness using financial data

## Objective
Predict an individual's creditworthiness using machine learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Features
- Data preprocessing
- Model training
- Credit score prediction
- Performance evaluation

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python credit_scoring.py
